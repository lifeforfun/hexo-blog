### 三角函数

#### 基本公式
> 邻边x、对边y、斜边r
$$
\sin{a}=\frac{y}{r}
\cos{a}=\frac{x}{r}
\tan{a}=\frac{y}{x}
\cot{a}=\frac{x}{y}
\sec{a}=\frac{1}{\cos{a}}=\frac{r}{x}
\csc{a}=\frac{1}{\sin{a}}=\frac{r}{y}
$$
$$\sin{0}=0$$
$$\sin^2{a}+\cos^2{a}=1$$
$$\tan{a}=\frac{\sin{a}}{\cos{a}}$$

| 三角函数 | 定义域 |
| -- | -- |
| $\sin{a}$ | R |
| $\cos{a}$ | R |
| $\tan{a}$ | $\{a\vert a\ne{k\pi}+\frac{\pi}{2},k\in Z\}$ |

#### 诱导公式
$$\sin(a+k*2\pi)=\sin{a}$$
> 上面公式cos、tan同样适用

$$\cos(-a)=\cos{a}$$
$$\sin(-a)=-\sin{a}$$
$$\tan(-a)=-\tan{a}$$
$$\cos[a+(2k+1)\pi]=-\cos{a}$$
$$\sin[a+(2k+1)\pi]=-\sin{a}$$
$$\tan[a+(2k+1)\pi]=\tan{a}$$
$$\cos(a+\frac{\pi}{2})=-\sin{a}$$
$$\sin(a+\frac{\pi}{2})=\cos{a}$$
$$\cos(-a+\frac{\pi}{2})=\sin{a}$$
$$\sin(-a+\frac{\pi}{2})=\cos{a}$$

> $x=\arcsin{y}$ 的意思是求正弦值为 $y$ 的弧度 $x$, 一般会限定 $x$ 的定义域.
> 此函数同样适用于cos、tan等
